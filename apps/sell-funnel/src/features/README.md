# Sell Funnel Features

Domain-specific features for the Sell Funnel vertical.

| Feature | Description | Phase |
|---------|-------------|-------|
| `funnel-builder/` | Drag-and-drop funnel creation | Phase 2 |
| `landing-pages/` | Template-based landing page builder | Phase 2 |
| `conversion-tracking/` | Event tracking, analytics, dashboards | Phase 2 |
| `ab-testing/` | A/B test framework for pages and funnels | Phase 2 |
| `ai-copy/` | AI-powered copywriting and design suggestions | Phase 2 |

Each feature is self-contained with its own components, hooks, and API routes.
